Detailed Description

System manager for saving and accessing external texture resources.

The TextureAssets is a singleton that allows the user to resuse saved external texture resources throughout their project.

Textures are saved via the Load() functions in the IonEngine::LoadResources() function. They can then be accessed by using the Get() functions and the associated key that the user defined upon loading.
